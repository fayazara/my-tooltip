# my-tooltip

### How to use
Add `v-tooltip` to any html element or a vue component & pass the tooltip message and position to it.
Example
```
<button v-tooltip="{ message: 'Right', position: 'right' }">Right</button>
```
#### Default values if nothing is passed
| message  | position |
|----------|---------|
| position | top     |

## Project setup
```
npm install
```
